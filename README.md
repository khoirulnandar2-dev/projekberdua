# L'Elite Café - Premium POS System

Sistem Kasir (POS) & Manajemen Inventori Premium untuk L'Elite Café. Dibuat dengan HTML5, Vanilla CSS, dan Vanilla JavaScript.

## 🚀 Fitur Utama
- **Dashboard Interaktif**: Statistik pendapatan, total transaksi, total menu terjual, dan peringatan stok menipis/habis beserta grafik tren penjualan.
- **Sistem POS (Kasir)**: Tab kategori menu, pencarian cepat, keranjang belanja real-time, dan kalkulasi otomatis subtotal, pajak (10%), serta total harga.
- **Kelola Inventori**: Pengaturan stok menu dan status ketersediaan secara real-time.
- **Laporan Penjualan**: Riwayat transaksi detail dan perhitungan pendapatan bruto/neto.
- **Multi-metode Pembayaran**: Mendukung QRIS (dengan QR Code otomatis), Kartu Debit, dan Tunai (dengan kalkulator kembalian otomatis).
- **Desain Premium**: Tampilan login glassmorphism modern, animasi halus, dan responsif.

## 🛠️ Cara Deploy ke GitHub Pages

Untuk menghosting aplikasi ini secara gratis di **GitHub Pages**, ikuti langkah-langkah mudah berikut:

1. **Buat Repositori Baru di GitHub**:
   - Masuk ke akun GitHub Anda.
   - Buat repositori baru (misalnya dengan nama `Projectberdua` atau `lelite-cafe-pos`).
   - Atur repositori menjadi **Public**.

2. **Upload Source Code**:
   - Upload semua file berikut ke root repositori Anda:
     - `index.html`
     - `style.css`
     - `app.js`
     - `images/` (folder gambar beserta isinya)

3. **Aktifkan GitHub Pages**:
   - Masuk ke tab **Settings** di halaman repositori GitHub Anda.
   - Di menu sebelah kiri, klik **Pages** (di bawah bagian *Code and automation*).
   - Di bawah **Build and deployment**, ubah Source menjadi **Deploy from a branch**.
   - Pada bagian **Branch**, pilih branch utama Anda (biasanya `main` atau `master`) dan folder `/ (root)`.
   - Klik **Save**.

4. **Akses Aplikasi**:
   - Tunggu sekitar 1-2 menit hingga proses build selesai.
   - Halaman web Anda akan dapat diakses secara publik melalui URL:  
     `https://<username-github>.github.io/<nama-repositori>/`
